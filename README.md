# Pharma-box-file

Pharma-Box Archivist: AI-Driven Document Orchestration

Problem: Pharmacy archives are flooded with unsorted digital scans.
Solution: A Multi-Agent system using the Model Context Protocol (MCP) to bridge an Enterprise Box storage with a Deep Learning classifier.

Key Features:

Automated Pipeline: Monitors Box folders for new medical uploads.

DL Classification: Uses a BART-based transformer model to categorize documents.

MCP Server Architecture: Decouples the AI logic from the storage layer, allowing for scalable integration.
